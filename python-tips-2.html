爬虫网页解析

  • 网页中文编码问题：在get后只提取content，而非text。content编码为unicode,可以直接print成为中文，而text有千奇百怪的编码,比如ISO-8859-1
  • BeautifulSoup 的标签提取：{}里面可以存在正则条件
  • find/findAll:一个／列表［］
  • 对于soup标签，可以用text提取出标签内所有文字
from BeautifulSoup import BeautifulSoup
import requests

for i in open('list2.txt').read().strip().split('\n'):
    try:
        data = requests.get('http://'+i).content
        #data = requests.get('http://'+i).text

        soup = BeautifulSoup(data)
        title = soup.find('title').text
        #title = title.decode('utf-8')
        #print requests.get('http://'+i).encoding
        print i+'\t'+title
    except:
        print i
